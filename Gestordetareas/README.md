# Gestor de Tareas

Aplicación web sencilla de gestión de tareas desarrollada con HTML, CSS y JavaScript.

## Descripción

Permite a los usuarios iniciar sesión, crear, buscar, completar y eliminar tareas. Las tareas se persisten en `localStorage`.

## Características

- Autenticación de usuario
- Agregar tareas con categoría
- Marcar tareas como completadas
- Eliminar tareas
- Búsqueda de tareas
- Estadísticas de tareas

## Tecnologías

- HTML5
- CSS3
- JavaScript (ES5/ES6)

## Uso

Abre `index.html` en cualquier navegador moderno.
